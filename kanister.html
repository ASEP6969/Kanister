Pengaturan Koneksi

Diperlukan sekali saja — token disimpan di penyimpanan pribadimu, tidak terlihat orang lain.

▾

Kalau tujuannya bagikan link ke orang lain, pilih Public — repo private membuat tautan unduh Release butuh login & akses, jadi tidak bisa dibuka bebas oleh penerima.

GitHub: buat token di Settings → Developer settings → Personal access tokens (scope repo), tempel di atas — lalu repo-nya tidak perlu dibuat manual, cukup ketik nama di kolom "Buat Repo Otomatis". Video disimpan sebagai lampiran Release, maksimal 2GB per file.
Google Drive: buat OAuth Client ID (tipe "Web application") di Google Cloud Console, aktifkan Drive API, tambahkan domain halaman ini ke Authorized JavaScript origins. Drive menampung file jauh lebih besar — cocok untuk rekaman raksasa.
